# AI Resume Analyzer

AI-powered Resume Analyzer built using n8n, Google Gemini, Google Sheets, and Gmail.

## Features

- Upload Resume PDF
- Extract Resume Content
- Analyze using AI
- Generate ATS Score
- Store Results in Google Sheets
- Send Analysis via Email

## Tech Stack

- n8n
- Google Gemini
- Google Sheets
- Gmail
- PDF Extraction
